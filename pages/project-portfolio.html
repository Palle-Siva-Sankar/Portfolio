About This Project

You are looking at it! This portfolio website was built to demonstrate my skills in combining modern web aesthetics with interactive 3D elements. The goal was to create an immersive experience that goes beyond a standard static resume, engaging visitors through motion and interactivity.

HTML5 CSS3 (Modern) Vanilla JavaScript Three.js Responsive Design

Technical Deep Dive

1. Architecture & Structure

The foundation of this website is clean, semantic HTML5. I prioritized a logical document structure using tags like <header>, <section>, and <article>. This not only helps with SEO but ensures the site is accessible.

2. Advanced Styling (CSS3)

  • CSS Variables: Extensive use of :root variables for colors and fonts allows for easy theming and consistent design changes across the entire site.
  • Glassmorphism: Used backdrop-filter: blur() and semi-transparent backgrounds to create modern, frosted-glass effects on cards and navbars.
  • Layouts: A hybrid approach using Flexbox for one-dimensional alignments (like navbars) and CSS Grid for two-dimensional layouts (like the skills and project grids).

3. Three.js 3D Integration

The immersive 3D footer isn't a video but a real-time rendered scene using Three.js (WebGL).

  • Scene Graph: Set up a scene, camera, and renderer. Loaded a GLTF model of an office desk to represent "work".
  • Lighting: Implemented Ambient and Directional lighting to create realistic shadows and depth.
  • Controls: Added OrbitControls to allow users to rotate and zoom into the scene, making the footer interactive.

4. Performance Logic (JavaScript)

To ensure 60fps performance even with 3D elements:

  • Intersection Observer: The 3D scene only renders when it is visible in the viewport. When you scroll away, the rendering loop pauses, saving your device's battery and CPU (check footer-3d.js).
  • Throttling: Scroll listeners and mouse-move effects (like the 3D card tilt) are throttled using requestAnimationFrame to prevent layout thrashing.
