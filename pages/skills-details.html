Core Technical Concepts

A breakdown of the fundamental topics I've mastered in my journey as a developer.

JavaScript Fundamentals

The logic that powers the dynamic web.

01. Variables & Data Types

Understanding let, const, and var, along with primitive types (String, Number, Boolean) and complex types (Objects, Arrays).

02. Functions & Arrow Functions

Writing reusable code blocks. Mastery of ES6 Arrow functions () => {} for cleaner syntax and lexical this binding.

03. DOM Manipulation

Interacting with the HTML structure. Selecting elements (querySelector), changing styles, and dynamic content updates.

04. Event Listeners

Handling user interactions like clicks, scrolls, and keypresses using addEventListener to make pages responsive.

05. Async Javascript

Managing operations that take time (like fetching data) using Promises and the modern async/await syntax.

06. ES6+ Features

Utilizing modern features like Destructuring, Template Literals, Spread/Rest operators to write efficient code.

07. Array Methods

data manipulation using high-order functions like .map(), .filter(), and .reduce().

08. Error Handling

Writing robust code using try...catch blocks to gracefully handle potential runtime errors.

09. LocalStorage & SessionStorage

Persisting user data in the browser so it survives page reloads and sessions.

10. Closures & Scope

Understanding how functions retain access to variables from their parent scope even after the parent function has closed.

HTML5 & CSS3

The structure and style of modern websites.

01. Semantic HTML

Using meaningful tags (header, nav, article) for better SEO and accessibility.

02. The Box Model

Understanding how margin, border, padding, and content interact to determine the size of elements.

03. Flexbox Layout

Creating one-dimensional layouts efficiently. Aligning items, distributing space, and handling responsiveness.

04. CSS Grid

Building complex two-dimensional layouts with rows and columns for robust page structures.

05. Responsive Design

Using Media Queries (@media) to ensure websites look perfect on desktops, tablets, and phones.

06. CSS Variables

Using custom properties (--main-color) for consistent theming and easier maintenance.

07. Animations

Bringing sites to life with keyframes, transitions, and transforms (translate, rotate, scale).

08. Accessibility (a11y)

Designing for all users, including those using screen readers, by using ARIA attributes and proper contrast.

09. SEO Best Practices

Optimizing content with meta tags, proper heading hierarchy, and alt text to improve search engine ranking.

10. CSS Preprocessors (SaaS)

Using tools like SASS/SCSS for nested rules, mixins, and more efficient style management.

MySQL & Database

Storing, retrieving, and managing data.

01. Relational Databases

Understanding tables, rows, and columns, and how structured data relates to each other.

02. CRUD Operations

The core functions: Create (INSERT), Read (SELECT), Update (UPDATE), and Delete (DELETE) data.

03. Keys & Indexes

Using Primary Keys to uniquely identify records and Foreign Keys to link tables together.

04. Joins

Combining rows from two or more tables (INNER JOIN, LEFT JOIN) based on a related column.

05. Filtering & Sorting

Refining queries using WHERE clauses, LIKE patterns, and ORDER BY.

06. Aggregation

Performing calculations on data sets using functions like COUNT(), SUM(), and AVG().

07. Normalization

Organizing data to reduce redundancy and improve data integrity (1NF, 2NF, 3NF).

08. ACID Properties

Ensuring reliable transactions: Atomicity, Consistency, Isolation, and Durability.

09. Stored Procedures

Saving complex SQL queries on the database server to be reused and executed efficiently.

10. Transactions

Grouping multiple operations into a single unit of work that either all succeed or all fail.

Full Stack Development

The big picture: Connecting Frontend to Backend.

01. Client-Server Architecture

How the browser (Client) requests resources and the Server processes and responds to them.

02. REST APIs

Building and consuming Application Programming Interfaces to exchange JSON data between systems.

03. Frontend Frameworks

Using libraries like React to build single-page applications with reusable components.

04. Backend Logic

Writing server-side code (Node.js/Java) to handle business logic, authentication, and routing.

05. Database Integration

Connecting the backend to the database to persist user data securely.

06. Version Control (Git)

Tracking changes, branching workflows, and collaborating with teams using Git and GitHub.

07. Deployment

Hosting applications on the web using platforms like Vercel, Netlify, or AWS.

08. Authentication & JWT

Securing applications using JSON Web Tokens to handle user login and session management.

09. Middleware

Functions that execute during the request-response cycle to handle tasks like logging or authorization.

10. Microservices

Breaking down large applications into smaller, independent services that communicate with each other.
