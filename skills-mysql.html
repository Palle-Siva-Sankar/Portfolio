MySQL & Database

Storing, retrieving, and managing data.

01. Relational Databases

Understanding tables, rows, and columns, and how structured data relates to each other.

02. CRUD Operations

The core functions: Create (INSERT), Read (SELECT), Update (UPDATE), and Delete (DELETE) data.

03. Keys & Indexes

Using Primary Keys to uniquely identify records and Foreign Keys to link tables together.

04. Joins

Combining rows from two or more tables (INNER JOIN, LEFT JOIN) based on a related column.

05. Filtering & Sorting

Refining queries using WHERE clauses, LIKE patterns, and ORDER BY.

06. Aggregation

Performing calculations on data sets using functions like COUNT(), SUM(), and AVG().

07. Normalization

Organizing data to reduce redundancy and improve data integrity (1NF, 2NF, 3NF).

08. ACID Properties

Ensuring reliable transactions: Atomicity, Consistency, Isolation, and Durability.

09. Stored Procedures

Saving complex SQL queries on the database server to be reused and executed efficiently.

10. Transactions

Grouping multiple operations into a single unit of work that either all succeed or all fail.
